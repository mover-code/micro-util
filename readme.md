# golang micro util  & go 微服务工具函数

## 部分只适用于 goZero 框架

### package 介绍

|名称|介绍|
|---|---|
|amz|亚马逊发送邮件服务|
|googleAuth| google身份验证器的生成与验证|
|ipfs| 本地ipfs服务上传api与nftstorage 上传|
|jwt| JWT  token 生成|
|mq| golang kafka 生产与消费|
|tools| 类型转换|
|translate| google 翻译api|
|request| 基于net/http 的封装http 请求|
|logger| 日志组件,目前直接写入至es,kibana 读取日志|
